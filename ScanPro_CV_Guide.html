📐 Gradient Analysis (Sobel)

We compute gradients to detect shadow boundaries using the Sobel operator.

Sobel Kernels

Gx (Horizontal)

-10+1 -20+2 -10+1

Gy (Vertical)

-1-2-1 000 +1+2+1
Gradient Magnitude

G = √(Gx² + Gy²)

💡 Retinex Decomposition

Based on the theory that an image is the product of illumination and reflectance.

Core Equations

Image = Reflectance × Illumination

Illumination ≈ GaussianBlur(L, kernel=31)

Reflectance = L / Illumination

Output = Reflectance × mean(Illumination)

# Python Implementation illumination = cv2.GaussianBlur(l_channel, (31, 31), 0) reflectance = l_channel / (illumination + 1e-6) output = reflectance * np.mean(illumination)

⚡ Canny Algorithm Steps

1. Gaussian Blur — Noise reduction with 5×5 kernel

2. Gradient Calculation — Sobel operators for Gx, Gy

3. Non-Maximum Suppression — Thin edges to 1px width

4. Double Thresholding — Strong/weak edge classification

5. Hysteresis — Connect weak edges to strong ones

6. Edge Tracking — Final binary edge map

📐 Stage 3: Contour Detection

From Edges to Document Boundaries

🔷 Douglas-Peucker Algorithm

Polygon approximation that simplifies contours while preserving shape.

Epsilon Calculation

ε = 0.02 × perimeter(contour)

# Polygon Approximation perimeter = cv2.arcLength(contour, True) epsilon = 0.02 * perimeter approx = cv2.approxPolyDP(contour, epsilon, True) # We need exactly 4 corners for a document if len(approx) == 4: corners = approx.reshape(4, 2)

📍 Corner Ordering Algorithm

Ensures consistent corner order: Top-Left → Top-Right → Bottom-Right → Bottom-Left

# Sort corners for consistent homography def order_corners(corners): # Sort by Y-coordinate sorted_by_y = corners[np.argsort(corners[:, 1])] # Top 2 and Bottom 2 top = sorted_by_y[:2] bottom = sorted_by_y[2:] # Sort each pair by X top_left, top_right = top[np.argsort(top[:, 0])] bottom_left, bottom_right = bottom[np.argsort(bottom[:, 0])] return [top_left, top_right, bottom_right, bottom_left]

🎯 Stage 4: Homography & Perspective Correction

The Mathematical Heart of Document Scanning

📊 Homography Matrix

A 3×3 transformation matrix that maps points between two planes.

The Homography Matrix H
h₁₁h₁₂h₁₃ h₂₁h₂₂h₂₃ h₃₁h₃₂h₃₃

8 Degrees of Freedom

h₃₃ is normalized to 1

✨ Point Transformation

How source points (x, y) are mapped to destination points (x', y').

Transformation Equations

x' = (h₁₁·x + h₁₂·y + h₁₃) / (h₃₁·x + h₃₂·y + h₃₃)
y' = (h₂₁·x + h₂₂·y + h₂₃) / (h₃₁·x + h₃₂·y + h₃₃)

🧮 Direct Linear Transform (DLT)

Algorithm to compute H from 4 point correspondences.

System of Equations (Ah = 0)

For each point correspondence (xᵢ, yᵢ) → (x'ᵢ, y'ᵢ), we get 2 equations:

[-x, -y, -1, 0, 0, 0, x·x', y·x', x'] · h = 0
[0, 0, 0, -x, -y, -1, x·y', y·y', y'] · h = 0

# OpenCV Implementation src_pts = np.array([[100, 80], [540, 60], [560, 400], [80, 420]], dtype=np.float32) dst_pts = np.array([[0, 0], [400, 0], [400, 500], [0, 500]], dtype=np.float32) # Compute homography matrix using DLT H = cv2.getPerspectiveTransform(src_pts, dst_pts) # Apply perspective warp warped = cv2.warpPerspective(image, H, (400, 500))

🔧 Morphological Operations

Structural Image Processing for Noise Removal

Dilation

Expands bright regions, closes gaps in detected edges.

dst(x,y) = max(src(x+x', y+y'))

Fill Edge Gaps

Erosion

Shrinks bright regions, removes small noise particles.

dst(x,y) = min(src(x+x', y+y'))

Remove Noise

Opening

Erosion → Dilation: Removes small bright spots.

cv2.morphologyEx(img, cv2.MORPH_OPEN, kernel)

Closing

Dilation → Erosion: Fills small dark holes.

cv2.morphologyEx(img, cv2.MORPH_CLOSE, kernel)
